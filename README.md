# Window-Pc-Clean
One click Clean window temp files

Windows System File Cleaning Tools

Temporary files (ယာယီဖိုင်များ) ဖျက်ခြင်း
Recycle Bin ရှင်းခြင်း
Cache files ရှင်းခြင်းဖြင့် GB များစွာ ချွေတာနိုင်သည်
ပိုလျှံနေသော ဖိုင်များ ဖယ်ရှားခြင်း
System ပိုမိုမြန်ဆန် လာစေခြင်း

✅ စနစ်စီမံခန့်ခွဲမှု ပိုမိုလွယ်ကူခြင်း
✅ အချိန်ကုန်သက်သာခြင်း
✅ မိမိဘာသာ မလုပ်တတ်သော နေရာများကို ရှင်းပေးခြင်း
